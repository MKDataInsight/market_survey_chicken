## Scénario :
Je suis data analyst chez La Poule qui Chante, une entreprise française d'agroalimentaire.
Notre activité principale est l’élevage et la vente de poulets sous le label “Poulet Agriculture Biologique”. 
Actuellement nosu commerçons en France, mais le PDG de l’entreprise souhaite évaluer la possibilité de se développer à l'international.
Pour l'instant, le champ des possibles est bien large : aucun pays particulier ni aucun continent n'est pour le moment choisi.
Mon rôle est donc de réduire notre cible et d'identifier les meilleurs lieux d'implantation.

## Objectifs et compétences
- **Objectif 1** : Exploiter un modèle d'apprentissage afin d'en apprendre davantage sur les données
- **Objectif 2** : Réaliser des analyses multivariées pour comprendre la corrélation entre les variables
- **Objectif 3** : Réduire la dimension du jeu de données pour optimiser le temps d'apprentissage des modèles
- **Objectif 4** : Sélectionner les variables pertinentes d'un modèle d'apprentissage

**Compétences développées** : Clustering, Prise en compte des enjeux contextuels, Justifier ma méthodologie, Adapter mon discours à un auditoire non technique, Effectuer des recommandations stratégiques, Recherche de données pertinentes


## Technologies utilisées
- **Python** : Pour la transformation des données, la visualisation des données et le clustering

## Structure du projet
Le projet est organisé comme suit :
